# Backup Directory

This directory stores export files created by the tool:

- `winget.json` — Exported Winget package list
- `chocolatey.config` — Exported Chocolatey package list (future)
- `scoop.json` — Exported Scoop package list (future)

These files are generated by the "Backup/Export" menu options and consumed by the "Restore/Import" options.
